    Objetivo geral:
  • Desenvolver serviços para web interoperáveis, abertos e flexíveis usando princípios RESTful e arquitetura em camadas por meio de frameworks e/ou bibliotecas.
    Objetivos específicos:
  • Desenvolver endpoints que podem ser acessados via web.
  • Conectar serviços web a clientes web.
  • Implementar persistência por meio de bibliotecas objeto-relacional.
  • Organizar a aplicação em camadas.
  • Desenvolver tratamento de exceções para webservices
