Aula 1 - Considerações

Avaliações

  • Portfólio: Deve incluir tudo que foi desenvolvido durante a Unidade Curricular durante o semestre. Prazo: 12/02/2025
  • Projeto funcional individual
  • Aplicação prática individual ao vivo ao lado do professor. Prazo: 22/01/2025

Linguagem de Programação

    Python: No mínimo a versão 3.11

Conhecimentos Requisitados

  • Criar ambiente virtual em Python
  • Extensões: python venv
  • Estar ambientado com sistema operacional Linux
  • Instalar pacotes
  • Nunca inserir código na pasta do ambiente virtual
  • Gerenciar o arquivo requirements.txt
  • Comandos para criar e ativar ambiente virtual: 
        python -m venv nome_do_ambiente_virtual
    nome_do_ambiente_virtual/Scripts/activate

Dependências para Instalar

  • flask
  • sqlalchemy
  • numpy (pip install numpy)

Sites Recomendados

  • PyPI (para seguir instruções de instalação de dependências)

Padrões

  • Variáveis e funções declaradas em inglês
  • Utilizar snake_case
